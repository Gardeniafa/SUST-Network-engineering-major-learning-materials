## Python数据分析 实验2 我的科大新闻  

运行方法：  
1、cd到当前目录  
2、命令`python manage.py runserver 0.0.0.0:80`运行服务  
3、如果端口被占用，更改端口即可，地址亦可从所有指定为回环地址  

如若不行，请检查是否具有Django环境，使用命令`pip install django`进行安装
